คำเตือน: ต้องเปิด Terminal แบบ Run as Administrator เสมอ

ไม่พบคำสั่งที่ตรงกับคำค้นหา

Pro Tips สำหรับช่าง

1. เครื่องอืด/ค้างบ่อย: เริ่มจาก sfc /scannow ตามด้วย DISM... เสมอ ก่อนตัดสินใจลง Windows ใหม่

2. เน็ตเชื่อมต่อแต่ใช้งานไม่ได้: มักแก้ได้ด้วยการ netsh winsock reset แล้วรีสตาร์ทเครื่อง

3. ไวรัสซ่อนไฟล์ใน Flash Drive: อย่าเพิ่ง Format! ใช้คำสั่ง attrib เพื่อเลิกซ่อนไฟล์ก่อน แล้วค่อยสแกนไวรัส

4. เปิดเครื่องไม่ติด (No Bootable Device): เข้า Recovery Environment แล้วใช้ชุดคำสั่ง bootrec เพื่อซ่อม Boot Sector

5. Windows Update ค้าง/Error: ให้หยุดบริการ wuauserv, ลบโฟลเดอร์ SoftwareDistribution แล้วเริ่มบริการใหม่

6. ก่อนเริ่มซ่อมเครื่องลูกค้า: สร้าง Restore Point ด้วย PowerShell เสมอ เพื่อความรับผิดชอบและป้องกันปัญหาตามมา
